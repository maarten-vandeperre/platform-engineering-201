# Platform Architecture Decision Records

This site documents key architecture decisions for the **People Platform** workshop system.

## Format

Each ADR captures:

- **Status** — proposed, accepted, or superseded
- **Context** — the problem or constraint
- **Decision** — what we chose
- **Consequences** — trade-offs and follow-up work
